## 🚀 Featured Project: LearnMint

📚 A powerful AI-based Study & News Dashboard built with cutting-edge tech stack.

> **Live:** [learn-mint-ai.vercel.app](https://learn-mint-ai.vercel.app) &nbsp;&nbsp;|&nbsp;&nbsp; **GitHub:** [LearnMint-Ai](https://github.com/mrgarvitoffice/LearnMint-Ai)

📌 **Features:**
- Gemini-powered content generation
- Auth with Firebase (Google, GitHub, Email)
- Realtime UI with React Query, ShadCN, framer-motion
- Animated, user-customizable themes
- NewsAPI via Newsdata.io
